FoodFactScanner — New Logo Options

Option 1: Shield Scanner

Shield (security) + scanner crosshairs + baby food jar with warning symbol. Conveys safety, scanning, and toxin detection.

Colors: Teal gradient shield, white scanner, yellow jar, red warning

Option 2: Bottle + Radar

Stylized baby bottle with scanning crosshairs, protected by an arc. Clean, modern, direct baby food safety association.

Colors: Teal bottle, yellow protective arc, white scan lines

Option 3:雷达检测

Circular badge with radar scanning lines, apple/food icon with leaf, and safety checkmark. Emphasizes health, scanning, and approval.

Colors: Teal badge, red apple arcs, green leaf, white checkmark

How to Use

Copy the SVG file to your project's src/assets/ folder and import it in your React components:

import logoImage from "@/assets/logo-new-1.svg";
// or whichever option you choose

<Logo src={logoImage} alt="FoodFactScanner" size="md" />

Update the Logo.tsx component to use the new SVG:

// In src/components/Logo.tsx
import logoImage from "@/assets/logo-new-1.svg";

export const Logo = ({ size = "md", className }) => {
  return (
    <img src={logoImage} alt="FoodFactScanner" className={...} />
  );
};
